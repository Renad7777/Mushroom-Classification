# 🍄 Mushroom Classification

## 📌 Overview

This project aims to classify mushrooms as **edible** or **poisonous** using Machine Learning techniques.
It helps in identifying potentially dangerous mushrooms based on their features.

---

## 🎯 Problem Statement

Mushroom classification is important for safety, as consuming poisonous mushrooms can be harmful.
This project builds a model that predicts whether a mushroom is safe to eat or not.
